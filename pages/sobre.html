Sobre a FarmaVida

Uma plataforma fictícia criada para demonstrar os serviços da AWS em um cenário real de e-commerce farmacêutico.

Nossa missão

A FarmaVida nasceu como projeto acadêmico com o objetivo de simular uma plataforma real de farmácia online, aplicando os conceitos de computação em nuvem da AWS em um contexto prático e dinâmico.

O projeto demonstra como construir uma aplicação escalável, segura e de alta disponibilidade utilizando os principais serviços AWS, desde a hospedagem até o banco de dados e distribuição de conteúdo.

Cada decisão arquitetural foi tomada pensando em segurança, performance e custo-benefício — os três pilares do Well-Architected Framework da AWS.

Arquitetura AWS

Serviços utilizados na infraestrutura da plataforma

Compute

Amazon EC2

Instâncias t3.medium para o servidor web. Auto Scaling Group configurado para até 4 instâncias conforme demanda.

Storage

Amazon S3

Armazenamento de imagens, assets estáticos e backups. Versionamento habilitado e lifecycle policies configuradas.

Database

Amazon RDS

MySQL 8.0 em Multi-AZ para alta disponibilidade. Backups automáticos diários com retenção de 7 dias.

CDN

CloudFront

Distribuição global de conteúdo com edge locations. Cache configurado para assets estáticos com TTL de 24h.

DNS

Route 53

DNS gerenciado com health checks e failover automático. Latency routing para melhor performance por região.

Security

AWS WAF + Shield

Proteção contra DDoS e ataques web. Regras gerenciadas da AWS e regras customizadas por IP e geolocalização.

Load Balancer

ALB

Application Load Balancer distribuindo tráfego entre instâncias EC2 com SSL/TLS terminado no balanceador.

Email

Amazon SES

Envio transacional de e-mails: confirmação de pedido, recuperação de senha e notificações de entrega.

Monitoring

CloudWatch

Monitoramento de métricas, logs centralizados e alarmes configurados para CPU, memória e latência.
